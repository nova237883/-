## 1. 产品概述

构建企业级数据可视化大屏系统，用于实时监控业务核心指标。系统支持多维度数据呈现、实时数据推送、自适应分辨率布局，适用于运维监控中心、业务指挥大厅等场景。

- **目标用户**：企业管理层、运营人员、数据分析师
- **核心价值**：通过可视化大屏直观呈现业务全貌，支持实时数据监控与决策辅助

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 系统预设 | 查看所有数据面板、管理数据源配置 |
| 访客 | 自动/免登录 | 查看大屏展示内容 |

### 2.2 功能模块

1. **大屏总览页**：KPI 指标卡、实时趋势折线图、排名柱状图、占比饼图、地理分布地图、仪表盘
2. **数据管理**：后端 RESTful API 提供数据接口
3. **实时数据推送**：WebSocket 连接，自动重连与心跳检测
4. **自适应布局**：基于 1920×1080 设计稿，支持任意分辨率缩放

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 大屏总览页 | KPI 指标卡 | 展示总销售额、订单量、活跃用户、转化率等核心指标 |
| 大屏总览页 | 实时趋势折线图 | 展示近 24 小时/7 天的业务数据变化趋势 |
| 大屏总览页 | 排名柱状图 | 展示各地区/品类的 Top10 排名对比 |
| 大屏总览页 | 占比饼图 | 展示各业务线/品类的占比分布 |
| 大屏总览页 | 地理分布地图 | 展示全国各区域的数据分布热力 |
| 大屏总览页 | 仪表盘 | 展示关键 KPI 完成率/达成率 |
| 大屏总览页 | 实时数据推送 | 通过 WebSocket 实时更新数据 |
| 大屏总览页 | 装饰边框 | CSS 渐变边框、扫描线动画、角标装饰 |

## 3. 核心流程

用户打开大屏页面后，系统自动加载各模块数据，并通过 WebSocket 建立实时连接，持续推送数据更新。

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝/黑背景（#0a1628、#1a2332），霓虹青（#00f0ff）、科技蓝（#1890ff）、霓虹紫（#7b61ff）点缀
- **按钮样式**：发光边框的扁平按钮，hover 时有霓虹光晕效果
- **字体**：数字使用等宽字体（DIN 或 Roboto Mono），中文使用系统无衬线字体
- **布局风格**：网格化卡片布局，每个图表模块带发光边框和装饰角标
- **图标风格**：线性科技感图标，与霓虹主题统一
- **动画效果**：数字滚动动画、图表入场动画、数据刷新微动效、扫描线背景动画

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 大屏总览页 | KPI 指标卡 | 半透明毛玻璃卡片、霓虹数字滚动、装饰边框 |
| 大屏总览页 | 趋势折线图 | ECharts 折线图、渐变填充区域、平滑曲线 |
| 大屏总览页 | 排名柱状图 | ECharts 柱状图、霓虹渐变柱体、横向排列 |
| 大屏总览页 | 占比饼图 | ECharts 饼图、环形样式、玫瑰图效果 |
| 大屏总览页 | 地理地图 | ECharts 地图、散点/热力效果、飞线动画 |
| 大屏总览页 | 仪表盘 | ECharts 仪表盘、多指针、分段色区 |
| 大屏总览页 | 装饰元素 | CSS 角标、扫描线、渐变边框、粒子背景 |

### 4.3 响应式设计

- 基于 1920×1080 设计稿，使用 CSS `transform: scale()` 实现全局缩放
- 监听窗口 `resize` 事件，自动计算缩放比例
- 图表组件独立监听尺寸变化，调用 ECharts `resize()` 方法
- 支持从 1366×768 到 4K 分辨率自适应

### 4.4 3D 场景（可选加分项）

- 使用 Three.js 或 ECharts GL 实现 3D 地图效果
- 地图区域悬停高亮，支持旋转交互
- 粒子系统背景增强科技感氛围
